# Employee Reimbursement Prediction using Multiple Linear Regression

## Overview
This project uses Multiple Linear Regression (MLR) to predict employee reimbursement values using income and benefits-related variables.

## Workflow
- Data Preparation
- Outlier Treatment
- Feature Selection
- Correlation Analysis
- VIF Analysis
- Multiple Linear Regression
- Interaction Effect
- Assumption Testing
- Model Evaluation

## Libraries Used
- pandas
- numpy
- seaborn
- matplotlib
- statsmodels
- scikit-learn

## Model Performance
The interaction model achieved the best performance with improved R-squared and lower prediction error metrics.
